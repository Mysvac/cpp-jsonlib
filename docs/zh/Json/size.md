# **Json.size**

```cpp
std::size_t size() const noexcept;
```

Returns the number of elements for container types, zero otherwise.

**Return Value**
- `Object`/`Array`: Current element count
- All other types (including `String`): `0`

**Exception Safety**  
No-throw guarantee

**Complexity** 
Constant

## Version

Since v3.0.0 .
