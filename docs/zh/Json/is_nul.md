# **Json.is_nul**

```cpp
constexpr bool is_nul() const noexcept;
```

Checks if the internal data is of `Null` type.

## Return Value
Returns `true` if the internal data is `Null`, otherwise `false`.

## Exceptions
No exceptions thrown.

## Time Complexity
Constant time O(1).

## Version

Since v3.0.0 .
