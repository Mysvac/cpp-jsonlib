# **Json.is_str**

```cpp
constexpr bool is_str() const noexcept;
```

Checks if the stored value is a string type.

**Returns**  
`true` if the data is a string, `false` otherwise

**Exception Guarantee**  
No exceptions will be thrown

**Complexity**  
Constant time operation O(1)

## Version

Since v3.0.0 .
