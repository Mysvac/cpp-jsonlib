# **Json.is_num**

```cpp
constexpr bool is_num() const noexcept;
```

Checks if the internal data is of type `Number`.

**Return value**:  
`true` if the data is numeric, `false` otherwise

**Exception safety**:  
No exceptions thrown

**Time complexity**:  
Constant time O(1)

## Version

Since v3.0.0 .
