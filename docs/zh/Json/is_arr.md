# **Json.is_arr**

```cpp
constexpr bool is_arr() const noexcept;
```

Determines whether the internal data is of type `Arr`.

## Return Value
`true` if the internal data is an `Arr`, `false` otherwise.

## Exception Safety
No-throw guarantee.

## Complexity
Constant time O(1).

## Version

Since v3.0.0 .
